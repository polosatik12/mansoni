# Mansoni

Социальная сеть Mansoni

**Автор:** Александр Караханов

## Технологии

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase

## Разработка

```sh
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

# Сборка проекта
npm run build
```

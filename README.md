 # Maisoni
 
 Социальная сеть Maisoni — современное мобильное приложение для общения.
 
 ## Технологии
 
 - React + TypeScript
 - Vite
 - Tailwind CSS
 - shadcn-ui
 - Capacitor (мобильное приложение)
 
 ## Локальная разработка
 
 ```sh
 # Установка зависимостей
 npm install
 
 # Запуск dev-сервера
 npm run dev
 ```
 
 ## Сборка
 
 ```sh
 npm run build
 ```
 
 ## Мобильное приложение
 
 ```sh
 # Android
 npx cap add android
 npx cap sync
 npx cap open android
 
 # iOS
 npx cap add ios
 npx cap sync
 npx cap open ios
 ```



## Цель
Создать эффект "тёмного прозрачного стекла" для модалки поиска с размытыми цветными бликами (как в референсе).

## Проблема
Сейчас `backdrop-blur` размывает серый/чёрный фон приложения — получается серое пятно без глубины и красоты.

## Решение
Сгенерировать фоновую картинку с цветными размытыми бликами (bokeh) и использовать её в модалке поиска.

---

## План изменений

### 1. Сгенерировать фоновое изображение
Использовать Lovable AI Image Generation для создания тёмного фона с:
- Глубокий тёмно-синий/чёрный градиент
- Размытые цветные блики (оранжевый, синий, фиолетовый)
- Мягкий bokeh эффект
- Разрешение подходящее для мобильных экранов

Prompt для генерации:
```
Dark glass background with soft colorful bokeh lights, deep navy blue to black gradient, 
blurred orange and blue spheres, abstract minimalist, smooth glassy texture, 
mobile wallpaper style, high quality, 9:16 aspect ratio
```

### 2. Сохранить изображение
**Файл:** `src/assets/glass-background.png`

### 3. Обновить ChatSearchSheet.tsx
**Файл:** `src/components/chat/ChatSearchSheet.tsx`

Изменения:
- Импортировать фоновое изображение
- Добавить его как background-image с overlay
- Сохранить backdrop-blur для дополнительного эффекта стекла
- Сделать overlay полупрозрачным чтобы фон просвечивал

```text
Структура слоёв:
1. Фоновое изображение (glass-background.png)
2. Полупрозрачный тёмный overlay (black/40)
3. backdrop-blur для контента под модалкой
4. Контент модалки
```

### 4. Обновить SheetOverlay
**Файл:** `src/components/ui/sheet.tsx`

Добавить проп `overlayClassName` чтобы можно было убрать тёмный overlay:
- По умолчанию: `bg-black/80`
- Для стеклянного стиля: `bg-transparent` или `bg-black/20`

---

## Результат

- Модалка поиска с красивым стеклянным эффектом
- Цветные размытые блики создают глубину и премиальный вид
- Контент под модалкой слегка просвечивает через стекло
- Соответствует референсу пользователя

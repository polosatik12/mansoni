

## План: Обновление фона страницы авторизации

Создам премиальный анимированный фон в цветах логотипа с эффектом "надутого зеркального шара" для аватара.

---

### 1. Новая цветовая палитра фона

Заменю текущий тёмно-фиолетовый градиент на цвета логотипа:

```text
Основные цвета:
┌────────────────────────────────────────────┐
│  #0066CC → #00A3B4 → #00C896 → #4FD080   │
│  (синий)   (бирюза)   (изумруд) (зелёный) │
└────────────────────────────────────────────┘
```

---

### 2. Анимированные переливающиеся шейдеры

Создам CSS keyframes для плавного перемещения и "дыхания" градиентных орбов:

| Элемент | Цвет | Анимация |
|---------|------|----------|
| Орб 1 | `#0066CC` (синий) | Движение по кругу 15s |
| Орб 2 | `#00A3B4` (бирюза) | Движение 18s с задержкой |
| Орб 3 | `#00C896` (изумруд) | Пульсация + движение 20s |
| Орб 4 | `#4FD080` (зелёный) | Вращение + масштаб 22s |

---

### 3. Зеркальный "надутый шар" для логотипа

Реализую эффект глянцевого пузыря как на референсе:

```text
┌─────────────────────────────────────┐
│    ╭──────────────────────╮         │
│   ╱  ✨ блик (белый)       ╲        │
│  │   ┌────────────────┐    │        │
│  │   │                │    │ ← внутренняя тень
│  │   │       M        │    │        │
│  │   │    (логотип)   │    │        │
│  │   └────────────────┘    │        │
│   ╲   фростовый gradient  ╱         │
│    ╰──────────────────────╯         │
│         ↑ внешнее свечение          │
└─────────────────────────────────────┘
```

Слои эффекта:
- **Внешнее свечение** — `box-shadow` с цветами логотипа
- **Стеклянный фон** — `backdrop-blur` + градиент от белого
- **Зеркальный блик** — `::before` с радиальным белым градиентом сверху
- **Внутренняя тень** — `inset box-shadow` для глубины
- **Анимация дыхания** — лёгкое увеличение/уменьшение шара

### 5. Результат

После внесения изменений страница авторизации будет иметь:
- Градиентный фон в фирменных синe-бирюзово-зелёных тонах
- Плавно перемещающиеся цветовые орбы с эффектом "дыхания"
- Глянцевый "пузырь" вокруг логотипа с зеркальным бликом
- Премиальный вид, соответствующий бренду Mansoni
